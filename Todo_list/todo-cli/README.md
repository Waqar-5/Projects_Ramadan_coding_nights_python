✅ To-Do List CLI App 📜
🚀 A simple, fast, and efficient command-line To-Do List manager built with Python.
Easily add, list, complete, and remove tasks – all from your terminal! 💻

🎯 Features
✔️ Add Tasks – Quickly add new tasks to your list.
✔️ View Tasks – See all tasks with their status (Pending ✅ / Completed ❌).
✔️ Mark as Completed – Mark tasks as done.
✔️ Remove Tasks – Delete a task from your list.
✔️ Persistent Storage – Your tasks are saved even after closing the app!
✔️ Easy-to-Use CLI – Simple command-based task management.

🛠️ Tech Stack
🔹 Python 🐍
🔹 Click Library (For CLI commands)
🔹 JSON Storage (Keeps your tasks saved)

📥 Installation & Setup
🔹 Step 1: Clone the Repository

git clone https://github.com/your-username/todo-list-cli.git
cd todo-list-cli
🔹 Step 2: Install Dependencies
Make sure you have Python 3.8+ installed. Then, install Click:

pip install click
🔹 Step 3: Run the App
Check available commands:

python todo.py --help
